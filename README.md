# CSS Classes

## 📌 Introduction

CSS classes are used to apply styles to multiple HTML elements in a reusable way. They help keep code organized, consistent, and easy to maintain.

Classes are defined in CSS using a dot (`.`) and are applied in HTML using the `class` attribute.

## 🎯 Purpose

* Reuse styles across elements
* Improve code readability
* Separate design from structure

## 🧠 Basic Example

```css
.box {
  color: blue;
}
```

```html
<div class="box">Hello</div>
```

## 📚 Summary

CSS classes are a fundamental concept in styling web pages, allowing developers to efficiently manage and reuse styles.
